# Инструкция по запуску торгового бота RSI + Bybit Futures

---

### Способ#1 (запуск в виртуальном окружении):
1. Установите uv
Если у вас MacOS - вместо 'pip' используете 'pip3'
```bash
pip install uv
```

2. Установите пакеты
```bash
uv sync
```

3. Заполните .env.dist, который находится в корне проекта и переименуйте его в .env

4. Заполните config.py, который находится в app/core/config.py

5. Запуск приложения
```bash
uv run -m app
```

---

### Способ#2 (запуск в докер контейнере):

1. Установите Docker с офф. сайта (https://www.docker.com/)
2. Заполните .env.dist, который находится в корне проекта и переименуйте его в .env
3. Заполните config.py, который находится в app/core/config.py
4. Запустите код:
```bash
docker compose up -d
```
